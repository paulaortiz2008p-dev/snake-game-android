# Atrapa el Ratón - Catch the Mouse Game

## Descripción
Juego móvil Android desarrollado en Kotlin. El jugador debe tocar los ratones que aparecen aleatoriamente en la pantalla antes de que desaparezcan.

## Características
- 🎮 Mecánica simple e intuitiva
- ⏱️ Modo de juego por tiempo (30 segundos)
- 📊 Sistema de puntuación en tiempo real
- 🎯 Dificultad progresiva
- 🎨 Interfaz amigable para estudiantes

## Requisitos
- Android Studio 4.0+
- Android SDK 21+
- Kotlin 1.5+

## Instalación
1. Abre Android Studio
2. Clona este repositorio
3. Abre el proyecto en Android Studio
4. Ejecuta en el emulador o dispositivo Android

## Controles
- Toca el ratón para ganar puntos
- Cada ratón vale 10 puntos
- El juego dura 30 segundos
